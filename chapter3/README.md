# Web Engineer In Rust Chapter3
## 要約
Rustの開発生産性を支える言語機能について抜粋して説明します。
型によるスレッド安全性の確保及びトレイトの活用がなぜ便利であると言えるのかを論点とします。
(厳密には、Rustの型システムにおけるスレッド安全性を議論するには、Sync, Sendというトレイトの理解も必要ですが、割愛してたりしてます。)
